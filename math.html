A bug travels from A to B along the segments in the hexagonal lattice pictured above.

The segments marked with an arrow can be traveled only in the direction of the arrow, and

the bug never travels the same segment more than once. How many different paths are there?

To solve this, we can split the diagram into 5 sectors, knowing that each arrow not filled

in is "restricted" in a way. I drew vertical partitions along the arrow verticals that were

only filled, and found the path possibilities for each section in turn. After connecting each

sector based on the answers I recieved, the problem resulted in an answer of 2400. This one

was strangely similar to a programming question . . .
